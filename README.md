# React Breadcrumbs 

[React][1] Component 

Demo at [opensourceprojects.robbestad.com][2]

## Installation

    % npm install react-breadcrumbs --save

## Usage


    var Breadcrumbs = require('react-breadcrumbs');

    MyComponent = React.createClass({
      render: function() {
         return (
           <div>
           	 <Breadcrumbs />
           </div>
        );
      }
    });
